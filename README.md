
# Hyprland Dotfiles!

Simple Dotfiles to install and enjoy!

First, you need to install all packages (there's in pacman-packages) after to clone this repo.


## Installation

To clone and move...

```bash
  git clone https://github.com/sklavier/dotfiles.git
  mv ./* ./.[!.]* ..
```

To install packages...

```bash
  pacman -S - < /home/$USER/pacman-packages.txt 
```
