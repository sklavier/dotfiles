## Installation

Clone mi repo and put it on User directory

```bash
  git clone https://github.com/EmilioBach/dotfiles.git
  cp -r ~/dotfiles/.config ~/.config
  cp -r ~/dotfiles/.local ~/.local
  cp ~/dotfiles/.zshrc ~/
```
That's all, enjoy :)
